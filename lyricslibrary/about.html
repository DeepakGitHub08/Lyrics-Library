About Us

We are team Coders.

We are a team of four IITB Undergrads. This is a website made by us as CS251 final project.
This a "lyrics library" which can be used as a hub for lyrics of Indian as well as non Indian languages.
Anyone can upload a lyrics here which will be reviewed by the admin and posted to the website
This website uses phpmyadmin, mysql, Apache server, PHP, html, css, bootstrap, Ajax and Javascipt.
We use Google language translate API provided by Center For Indian Language Technology, IIT Bombay to convert English to Indian languages real time
The fully documented source code for the website will be available Github soon. Feedbacks are welcome!
